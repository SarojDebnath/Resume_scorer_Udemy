TO RUN THIS CODE BASE:

Follow these steps

1. Create a python environment
2. In a terminal, install git (conda install git) and then enter: git clone https://github.com/SarojDebnath/Resume_scorer_Udemy.git
3. cd Resume_scorer_Udemy
4. pip install -r requirements.txt
5. Next in the same terminal, run uvicorn api.main:app --reload
